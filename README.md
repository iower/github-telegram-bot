# github-telegram-bot
Notify about GitHub commits in Telegram

![image](https://github.com/iower/github-telegram-bot/assets/35522011/5a62108a-e5a9-4ad1-aa95-1d72aea4f1b5)


## Requirements
- nodejs16+

## Installation
> npm i

## Config
> mv config.example.json config.json

And edit `config.json`

## Run
> npm run bot
